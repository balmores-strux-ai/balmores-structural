# BALMORES STRUCTURAL

Prototype full-stack website that reads the trained 5000-model structural brain and turns free-text building input into:

- structured assumptions
- instant structural predictions
- a live 3D browser model
- grouped beam/column recommendations
- chat-style follow-up conversation
- ETABS verification job placeholder

## Stack

- **Frontend:** Next.js (App Router) + React + React Three Fiber
- **Backend:** FastAPI + PyTorch
- **Model:** `backend/models/etabs_parametric_structural_brain.pt`

## Quick Start

### Backend

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open:

- Frontend: http://localhost:3000
- Backend docs: http://localhost:8000/docs

### Environment

For the frontend to reach the backend when running locally, it uses `http://localhost:8000` by default. Override with `NEXT_PUBLIC_API_URL` if your backend runs elsewhere.

## Prototype Notes

- The parser is intentionally forgiving and assumption-driven.
- Follow-up prompts merge into the current project state.
- Results are generated by the trained brain bundle plus deterministic post-processing for prototype UX.
- ETABS verification endpoint is a stub placeholder to be connected later to your real ETABS worker.
